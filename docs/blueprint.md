# **App Name**: FocusFlow

## Core Features:

- Task Dashboard: Display today's tasks, upcoming deadlines, study timer, and daily progress.
- Smart Task Manager: Enable users to add, edit, delete tasks, set priority, add subject tags, due date & reminders, and mark as completed.
- Study Timer: Implement a Pomodoro-style timer (25 min focus / 5 min break) with custom timer option and session counter.
- Goal Tracker: Track weekly and monthly goals with progress displayed as a percentage.
- Daily Planner View: Show a timeline (6 AM â€“ 11 PM) with drag and drop scheduling functionality.
- Analytics Section: Display study hours per subject, completed tasks graph, and productivity streak counter.
- Intelligent Scheduling: Use generative AI as a tool to analyze past study habits and suggest optimal study times, highlighting overdue tasks. Show motivational quotes on dashboard.

## Style Guidelines:

- Primary color: Soft lavender (#D0BFFF), for a calm and focused feel.
- Background color: Light gray (#F0F0F5), creating a clean and distraction-free environment.
- Accent color: Muted blue (#A0C4FF) to highlight important tasks and CTAs.
- Body and headline font: 'Inter' for a modern and clean aesthetic.
- Minimalist and distraction-free design to promote focus.
- Smooth transitions and subtle animations to enhance user experience.
- Use simple and consistent icons for easy navigation.
